# Tabu Search algorithms for the vertex k-center problem

stm_tabu_search.cpp implements a basic attribute-based short-term memory strategy for the vertx k-center problem.

ltm_tabu_search.cpp implements a basic attribute-based long-term memory strategy for the vertx k-center problem.
